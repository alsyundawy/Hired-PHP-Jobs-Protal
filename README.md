## HIRED
Hired is an open-source PHP Jobs Portal. This is not a "we have all sorts of features that you will never use" kind of project.Â  This is a simple portal with the basic features, so you can customize and build on top of it quickly.

Just replace all the files and access `index.php`, the installer will take care of the updates.
<br><br>


## REQUIREMENTS
1) Not extensively tested, but should work with at least PHP 7.4.
2) PHP MYSQL PDO extension.
3) Apache server with MOD REWRITE enabled.
4) "Grade A" browser.
